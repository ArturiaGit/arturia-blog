## ✨ 核心架构 (Core Features)

* **极致 UI/UX 视觉：** 废弃传统暖色调，全站采用【晨曦白 × 冰蓝 × 霓虹青】的极客冷色调，搭载全局赛博毛玻璃拟态面板。
* **全息数据列阵：** 彻底重构分类页与归档页，搭载悬浮动态推进与霓虹光效。
* **沉浸式环境光引擎：** 底层植入无缝交叉渐变的双层画布壁纸引擎，支持跨页面状态持久化记忆，并在文章页智能感知专属封面。
* **极客阅读排版系统：** 专为 Markdown 打造的文章渲染规范，融入 `JetBrains Mono` 等宽代码字体与分层高亮体系。
* **防碎裂响应式布局：** 彻底修复 Flex 弹性盒溢出陷阱，实现标签胶囊内部的优雅单行截断。
